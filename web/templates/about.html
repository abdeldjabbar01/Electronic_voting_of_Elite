Electronic Voting System

Who We Are

We are a group of five third-year engineering students at ENSTA Alger, specializing in Artificial Intelligence and Security Systems. This project is part of our Cryptology course under the supervision of Mrs. Souad KHERROUBI.

Project Goal

Create a secure, anonymous, and verifiable electronic voting system based on the blind signature protocol studied in class. The system guarantees:

Voter Anonymity

No single entity can link a vote to its voter.

Vote Integrity

Every ballot is signed blindly by an administrator and encrypted for a counter, with TTH integrity checks.

Double-Voting Prevention

The commissioner marks each voter's N1 code as used.

End-to-End Verifiability

Voters can later check that their N2 code appears in the published results.

Universal Verifiability

Anyone can independently verify that all votes were counted correctly.

Eligibility Verifiability

Only registered voters with valid credentials can cast votes in the election.
